# Welcome to taucoinj

# About
taucoinj is a pure-Java implementation of the Taucoin protocol.


# Running TAUCOINJ

##### Running from command line:
```
> git clone https://github.com/Tau-Coin/taucoinj
> cd taucoinj
> gradlew run [-PmainClass=<sample class>]
```
